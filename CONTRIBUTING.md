# Contributing

Thank you for your interest in improving this project.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Add or update documentation as needed.
5. Submit a pull request with a clear description.

## Coding Style

- Follow PEP 8 for Python code.
- Use descriptive variable names.
- Keep notebooks clean and well-documented.
- Avoid altering existing notebook logic unless necessary.

## Pull Requests

Please keep pull requests focused and include a short summary of the change.

## Issues

Use GitHub Issues to report bugs, suggest improvements, or ask questions.
