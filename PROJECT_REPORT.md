# Project Report

## Abstract

This project presents a machine learning solution for handwritten digit recognition using the MNIST dataset. The work focuses on building, training, and evaluating deep learning models in a structured and explainable way.

## Problem Statement

Handwritten character recognition is a foundational computer vision task with applications in document processing, banking, and OCR systems. The goal is to classify handwritten digits accurately using deep learning.

## Objectives

- Train a model that can accurately classify handwritten digits
- Compare baseline ANN and CNN approaches
- Document the workflow clearly for portfolio presentation

## Dataset

The project uses the MNIST dataset, which contains 28x28 grayscale images of handwritten digits from 0 to 9.

## Methodology

The workflow includes data loading, normalization, reshaping, EDA, model building, training, evaluation, and result visualization.

## Model Architecture

The CNN model uses convolutional and pooling layers followed by dense layers for classification into 10 classes.

## Training

Training was completed using the Adam optimizer and categorical cross-entropy loss with validation support.

## Evaluation

Model performance was assessed through accuracy, loss, and classification reports.

## Results

The project achieved strong classification performance and produced a polished GitHub portfolio-ready repository.

## Conclusion

This project demonstrates a complete machine learning pipeline for handwritten digit recognition and serves as an excellent portfolio piece.
