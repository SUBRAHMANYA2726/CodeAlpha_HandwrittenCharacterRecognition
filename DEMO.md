# Demo Guide

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open the notebook:
   ```bash
   jupyter notebook notebooks/CodeAlpha_HandwrittenCharacterRecognition.ipynb
   ```
3. Run the cells in sequence.

## Expected Output

- Visualizations in the notebook
- Training and validation accuracy/loss plots
- Metrics saved in the outputs folder
- A saved model artifact in the saved_models folder

## Prediction Examples

Prediction examples and sample output files can be placed in the [outputs/predictions](outputs/predictions) folder.
